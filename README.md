## Dependencies

```bash
php: >=7
yii2: *
yii2-bootstrap5: *
```


## Run Locally

Clone the project

```bash
  git clone https://github.com/linlaz/yii-mazer.git
```

Go to the project directory

```bash
  cd yii-mazer
```

Install dependencies

```bash
  composer install
```

Start the server

```bash
  php yii serve
```
